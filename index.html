BLE file transfer · pixl.js · v0.6
Web Bluetooth isn't available here. This page must run in Chrome or Edge (desktop or Android) over https or localhost. It will not connect inside an embedded preview — download the file and open it in Chrome, or host it (e.g. GitHub Pages).
Before you connect: on the device, set Settings → Sleep Timer → Off (it sleeps in ~15s and drops the link), then open the BLE File Transfer screen. The device pairs as Pixl.js. Filenames are capped at 47 bytes and full paths at 63 bytes — longer names are rejected here so they don't get silently truncated.
